# 内墙卡牌猜谜游戏

这是一个简单的卡牌猜谜游戏，玩家需要根据显示的卡牌猜测对应的管理人首字母缩写。

## 游戏规则

1. 游戏开始后，界面会随机显示一张卡牌
2. 玩家需要在输入框中输入对应管理人的首字母缩写
3. 答对后会显示分享界面，可以分享游戏
4. 答错后可以选择重试或者换下一张卡牌
5. 游戏结束条件是所有卡牌轮换完毕

## 答案规则

卡牌对应的答案是管理人的首字母缩写：

- 1-5号卡牌: kd
- 6-10号卡牌: pfh
- 11-17号和45号卡牌: ps
- 18-24号卡牌: mh
- 25-31号卡牌: ms
- 32-35号和46号卡牌: lq
- 36-44号卡牌: hf

## 运行游戏

1. 确保卡牌图片放在正确的位置：
   - 将所有卡牌图片（1.png, 2.png, ...）放在 `assets` 目录下
   - 或者放在 `assets/CodeBubbyAssets/28_154` 目录下

2. 运行本地服务器：
   ```
   python server.py
   ```

3. 在浏览器中访问：
   ```
   http://localhost:8000
   ```

## 文件结构

- `index.html`: 游戏主界面
- `css/style.css`: 样式表
- `js/cardManager.js`: 卡牌管理器
- `js/game.js`: 游戏逻辑
- `js/ui.js`: UI交互
- `server.py`: 本地服务器
- `assets/`: 卡牌图片和其他资源

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Python (用于本地服务器)
