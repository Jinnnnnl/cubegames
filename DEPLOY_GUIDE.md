# 腾讯云Studio部署指南

## 本次更新内容
- 修复了多卡牌显示时底部提示词被截断的问题
- 优化了移动端响应式布局
- 改进了卡牌容器的高度自适应

## 需要更新的文件
1. `css/style.css` - 主要修改文件，包含响应式布局优化
2. `index.html` - 游戏主页面
3. `js/game.js` - 游戏逻辑
4. `js/cardManager.js` - 卡牌管理
5. `js/ui.js` - UI交互
6. `assets/` - 资源文件夹（如有新增）

## 部署步骤

### 方法1：通过腾讯云Studio界面上传
1. 登录腾讯云Studio
2. 打开你的项目工作空间
3. 将以下文件上传/替换：
   - `css/style.css`
   - `index.html`
   - `js/` 文件夹下的所有文件
   - `assets/` 文件夹（如需要）

### 方法2：通过Git同步（推荐）
```bash
# 如果项目已连接Git仓库
git add .
git commit -m "修复多卡牌显示问题，优化移动端布局"
git push origin main

# 然后在腾讯云Studio中拉取更新
git pull origin main
```

### 方法3：压缩包上传
1. 将整个项目打包为zip文件
2. 在腾讯云Studio中解压替换

## 验证部署
部署完成后，请测试：
1. 桌面端：多张卡牌时底部提示词是否正常显示
2. 移动端：响应式布局是否正常
3. 游戏功能：确保所有交互正常工作

## 主要修改说明
- 卡牌容器从固定高度改为自适应高度
- 添加了滚动支持，确保内容不被截断
- 优化了移动设备的显示效果
